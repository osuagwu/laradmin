The Base template is  for advanced users. The template does not include sidebars or container class or sections  class. It gives an opportunity to design a page structure from WP editor.  It hover includes the header and the footer. When using this template, wraps the content with  correct container class, and  if required the section class of choice. Example: 
<section class="section section-default ">
        <div class="container">
           CONTENT
        </div>
    </section>
Where the container class in the header and footer included by the template can be changed to container-fluid class by setting the wide_screen custom field to on.
