The hero template

This hero template allows you to create a beautiful and simple page which has a hero image. The these are the parts of a hero page:
  • The hero image is a featured image of the corresponding page in WP
  • The first H1 on the page is assumed to be the title of the hero. A tagline for the title can be created my add a small element tag in the H1 tag. Thus: 
    <h1>This is the hero title
    <small class="text-white">Small bit of the title which could serve as a tag line</small></h1>
  • Any of the following can be used to create buttons inside the hero image: 
    • hero_url url="http://www.itv.com" text="See the ITV" 
    • hero_route name=user-profile text="User profile" 
    •  hero_url url="http://www.bbc.co.uk" text="See the BBC"
  • Any other content on the page will be placed right after the hero image.
